# Agent Wallets

The dashboard exposes a managed wallet workspace for operators who want a smoother paid-call flow.

## What you can do

- create a new agent wallet
- import an existing funded wallet
- generate and rotate a connect string
- test a live paid call
- inspect wallet activity
- manage funding actions

## Best use case

Use the agent wallet flow when you want a repeatable operator-friendly payment path without building your own raw x402 signing loop for every call.

## Important boundary

The public docs can describe the workflow and the public gateway behavior, but they do not document the private fields, storage model, or internal token handling behind the managed wallet feature.
