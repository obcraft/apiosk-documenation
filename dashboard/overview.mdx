# Dashboard Overview

The dashboard is the operator workspace around the public gateway.

## Main sections

### APIs

Create listings, manage endpoint pricing, inspect payments, and copy the public gateway base URL for your API.

### Agent Wallets

Manage funded agent wallets, connect strings, live wallet activity, and payment-related actions from one workspace.

### Usage And Revenue

Inspect paid request volume, current wallet spend, revenue earned, and payout state through the dashboard views.

## Public-safe framing

This docs project describes what the dashboard lets an operator do. It does not document dashboard-only secrets, private storage design, or internal gateway implementation details.
