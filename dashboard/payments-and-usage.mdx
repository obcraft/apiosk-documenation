# Payments And Usage

Once an API is live, the dashboard gives operators visibility into the commercial side of traffic.

## Typical views

- paid request volume
- total calls
- revenue by API
- wallet spend
- payout state
- recent payment activity

## API-level workflow

For an individual API, operators typically use:

- setup
- endpoints
- payments
- gateway

These views help you move from publishing to pricing to ongoing monitoring.
